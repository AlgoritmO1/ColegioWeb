## Pagina web colegio
Este proyecto es el mismo que se hizo para python solo que se realiza en JSP para que sea una página web.
*******************************************************************
**********************SISTEMA DE NOTAS*****************************
*******************************************************************

-> El sistema de notas consiste en un sistema especializado para
que los estudiantes y profesores, puedan acceder mas facilmente a
sus notas y materias inscritas en una institución educativa.

Antes de acceder al sistema, tendrá un usuario y contraseña
d eacuerdo al rol que desempeña, cada rol tiene unas opciones
distintas y seran mostradas acontinuacion.

********************************************************************
*******************ROL DE ADMINISTRADOR*****************************
********************************************************************

1. Inicio
2. Estudiante
3. Profesor
4. Materia
5. Matricula

ANTES DE ENTRAR HAY QUE LOGUEARSE CON: 
Usuario: Admin
Contraseña: 12345

-> En la opcion de inicio, lo retorna al inicio de la pagina.

-> La opcion de estudiantes tiene a su veces las siguientes
opciones:

- Crear estudiantes -> Que se usa para registrar un estudiante
y la lista.
 - Datos guardados -> Es para revisar los estudiantes que aparecen
inscritos en la institución.

-> La opcion de profesor tiene las siguientes opciones:

- Crear profesor -> Esta opcion se usa para llenar los campos
correspondientes al profesor, con su respectiva materia a dictar.

- Datos guardados -> Se revisa la lista de profesores que se han
inscrito hasta el momento.

-> La opcion de materia cuenta con las siguientes opciones:

- Crear materia -> Crea una materia con sus respectivos campos

- Datos guardados -> Lista las materias guardadas.

- Materia con mas matriculas -> Muestra las materias que han tenido
mas matriculas.

- Materia con mas profesores -> Muestra a la materia que tiene a
mas profesores dictandola.

- Materia con mejor nota -> Muestra la materia con la mejor nota.

-> La opcion de matricula cuenta con lo siguiente:

- Crear matricula -> Crea matriculas con sus respectivos campos
asignandola a un profesor, un estudiante y una materia.

- Datos guardados -> Muestra la lista de matriculas que se 
han inscrito hasta el momento.

- Mejor promedio -> Muestra el mejor promedio que hay en las
matriculas listadas.

- Estudiantes por materia -> Muestra los estudiantes que hay en 
una materia especifica.

- Nota mas alta -> Muestra la nota mas alta en las materias listadas

- Nota mas baja -> Muestra la nota mas baja en las materias listadas
